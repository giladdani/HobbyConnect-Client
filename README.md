# HobbyConnect-Client
## HobbyConnect is a React.js web application designed to bring people together.
Whether you know them or not, many people enjoy the same hobbies as you do and will be more than happy to have you engage them together!
What about you? Do you plan a fun activity and are looking for more participants to join? Post about it in HobbyConnect and find them!

## HobbyConnect is written in TypeScript and the React.js library while integrating with a RESTful API written in Node.js at [HobbyConnect-Server](https://github.com/giladdani/HobbyConnect-Server)
